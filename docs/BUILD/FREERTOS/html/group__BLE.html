Description

The nina_b112 driver provides support for for data transfer using a BLE module, here a NINA B112 BLE module. This module is interfaced on GAPPOC through UART.

The BLE driver provides support for wireless data transfer between the host and an external peripheral such as a smartphone, using Bluetooth connection.
