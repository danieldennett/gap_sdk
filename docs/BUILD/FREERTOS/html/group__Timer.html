Parameters
timerTimer to set.
time_usTimer value in us.
one_shotTimer mode : one-shot or continuous.
Note
one_shot = 1, one shot timer. one_shot = 0, continuous mode.
This function disables a timer if one is already enabled, and reconfigure it.
